## How to Use

When the OSINT Agent receives a subject for credential lookup:

1. Identify subject's state of employment at time of incident
2. Look up the correct board URL in the tables above
3. Run `web_fetch` against the board's credential lookup page
4. If state board URL fails or has no public lookup → fall back to national source
5. Log each lookup attempt with timestamp, source URL, and result
6. Apply False Positive Guard — NOT FOUND ≠ NOT CERTIFIED

If a URL above is broken or a state has moved its board:
- Search: `<state name> EMS licensing board` or `<state name> POST commission`
- Update this router with the corrected URL
- Note the date of update
